# Synent Technologies Internship - Task 2
# Iris Data Visualization Dashboard

## Problem Statement

The objective of this project is to visualize patterns and relationships in the Iris dataset using various charts and interactive controls.

## Dataset Details

Dataset Name: Iris Dataset

Source: Scikit-Learn

Records: 150

Features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target Classes:
- Setosa
- Versicolor
- Virginica

## Approach

1. Loaded Iris dataset using sklearn.
2. Created an interactive Streamlit dashboard.
3. Generated:
   - Bar Chart
   - Histogram
   - Scatter Plot
   - Box Plot
   - Correlation Heatmap
4. Added feature selection controls for users.

## Results

- Petal features clearly differentiate species.
- Setosa is easily separable from other classes.
- Strong positive correlation exists between petal length and petal width.
- Interactive dashboard allows dynamic feature comparison.
